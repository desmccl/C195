•  directions for how to run the program

To run the program
1. Make sure you are connected to the database
2. Open the Test2 project in the IDE
3. Make sure you are on the main class, it is called HelloApplication.java
4. Click the green "Play" icon on the top right of the screen and wait a moment for the project to build
5. Once it is finished building, the application will display the login page
